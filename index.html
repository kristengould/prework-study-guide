Prework Study Guide

HTML

  • Provides content and basic framework for all pages on the web.
  • Made up of text, image, header, and list elements.
  • The head element contains information (metadata) about the webpage.
  • The body element represents the visible contant shown to the user.
  • Metadata is data about our webpage that the browser uses to effectively serve our webpage users.

CSS

  • Defines the clolors, aesthetics, visual layout, and organization of a webpage.
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an elemnt.
  • Inline CSS: apply to a specific HTML tag to style a particular page element only.
  • Internal Style Sheet: embeds an entire stylesheet directly in the HTML file.
  • External Style Sheet: allows evelopers to keep all their CSS rules in a seperate file.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git add -A: commits all files in the working branch
  • git commit -m: commits changes made with a description of what was changed
