HTML

  • Provides content and basic framework for all pages on the web.
  • Made up of text, image, header, and list elements.
  • The head element contains information (metadata) about the webpage.
  • The body element represents the visible contant shown to the user.
  • Metadata is data about our webpage that the browser uses to effectively serve our webpage users.

CSS

  • Defines the clolors, aesthetics, visual layout, and organization of a webpage.
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an elemnt.
  • Inline CSS: apply to a specific HTML tag to style a particular page element only.
  • Internal Style Sheet: embeds an entire stylesheet directly in the HTML file.
  • External Style Sheet: allows evelopers to keep all their CSS rules in a seperate file.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git add -A: commits all files in the working branch
  • git commit -m: commits changes made with a description of what was changed

JavaScript

  • Allows users to interact with websites.
  • Variables: containers that store values; declare a variable with the 'var' keyword, followed by the name of the variable.
  • Operator: a mathematical symbol that produces a result based on two values (or variables)
  • Conditionals: code sturctures used to test if an expression returns true or not
  • Functions: a way of packaging funcionality that you wish to reuse
  • Events: code structures that listen for activity in the browser, and run code in response
  • Symbol: a unique and immutable primitive value that may be used as the key of an Object property
  • Objects: the only mutable values, can be seen as a collection of properties
  • Arrays: regular objects for which there is a particular relationship between integer-keyed properties and the length property
  • JSON: a lightweight data-interchange format that builds universal data structures that can be transferred between different environments and even accross languages
  • Control flow is the order in which a computer executes code in a script.
